# HverdagsHelt - Team 3

The application will reload on source changes.

## Client: run tests and start

From the top-level repository folder:

```sh
cd client
npm install
npm test
npm start
```

## Server: run tests and start

Prerequisite: mysql-server installed locally

From the top-level repository folder:

```sh
cd server
npm install
npm test
npm start
```

## Open application

http://localhost:3000

force change
